SOBRE OS ESTUDIOS-ADRIAN

Foi criado em 2023 para entender como funciona o estilo de criar jogos usando variáveis, e mostrar que é possível criar projetos sem utilizar códigos de criação de jogos. No entanto, para criar um projeto de um jogo é mais complexo, pois tem que ser levado em consideração eventos que podem ser feitos e estilos a serem usados, como criar tabelas de valores diferentes que mostrem onde esses eventos acontecem.
